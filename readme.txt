D is for dark - Espresso theme

Requirements
=============
This theme has been designed to work with Espresso 2.0 and may not work with earlier versions.

How to use
=============
Copy the 'disfordark.css' file to your Espresso themes folder and activate.

Espresso's themes folder can be easily opened while in Espresso by accessing preferences > colors and clicking 'reveal in finder' which will open Espresso's themes folder. Add the zip files content to this folder and select the theme in Espresso's preferences window to activate it.
